# Reportes y Figuras

En esta carpeta se almacenan los resultados del análisis:

- **figures/**: Gráficas exportadas (PNG, JPG, PDF) generadas en los notebooks o scripts.

Las figuras generadas automáticamente suelen estar ignoradas en el `.gitignore` para evitar saturar el repositorio con archivos binarios, a menos que sean necesarias para la documentación.
